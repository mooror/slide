This package is a defualt file for creating extras for modx revolution.

The tutorial to go along with this file can be found at:
http://rtfm.modx.com/revolution/2.x/case-studies-and-tutorials/developing-an-extra-in-modx-revolution

I replaced all of the words doodles partianing to the code with -extra name so that you can know where the name of your 
extra goes in the code.
